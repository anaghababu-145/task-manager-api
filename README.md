# Task Manager API

A RESTful backend API for managing tasks.

## Tech Stack

- Node.js
- Express
- TypeScript
- MongoDB
- Mongoose
- JWT Authentication
- bcrypt

## Features

- User Registration
- User Login
- JWT Authentication
- Protected Routes
- CRUD Task APIs
- Tasks linked to authenticated users

## API Endpoints

POST /api/auth/register  
POST /api/auth/login  

GET /api/tasks  
POST /api/tasks  
PUT /api/tasks/:id  
DELETE /api/tasks/:id
